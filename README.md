# Science Bee Study

Practice tools and study resources for the National Science Bee.

## Contents

- `science_bee_practice.html` — Interactive practice tool for Science Bee questions

## Quick Start

1. Download [`science_bee_practice.html`](https://github.com/steadycoding/science-bee-study/blob/main/science_bee_practice.html)
2. Double-click the file to open it in your browser — no installation required
3. Everything is self-contained in that single file

## How to Use

1. **Select question sets** using the checkboxes on the start screen
2. Optionally filter by **category** (Biology, Chemistry, Physics, etc.)
3. Click **START + LISTEN** to begin — questions will be read aloud
4. Press the **BUZZ** button (or Space/Enter) as soon as you know the answer
5. **Speak your answer** — the app will try to grade it automatically
6. If the app misses your answer, use the **✓** (correct) or **✗** (wrong) buttons to mark it manually
7. After finishing, review your results in the **end screen** table broken down by category

## Tips for Best Results

- Use **Chrome** or **Edge** for the best speech recognition experience
- Sit in a **quiet room** close to the microphone
- **Close the tab** when you're done practicing (see Known Limitations below)
- Encourage answering **early in the question** to earn bonus points for buzzing before the final clue

## Why It's Useful

- Goes through questions faster than manual flashcards
- Bonus points for early answers incentivize knowing the material deeply
- Category breakdown at the end makes it easy to identify which topics to focus on

## Known Limitations

1. **Speech recognition sometimes misses the answer.** Just use the ✓/✗ buttons to manually mark whether the answer was correct, then review missed questions at the end.

2. **Microphone stream stays open during the session.** To improve recognition accuracy, the app keeps the microphone connection to Google's speech recognition servers open while the tab is open. Keep the app open only in the room where the student is practicing, and **close the tab when done** to stop sending audio to Google's servers.

3. **Browser repeatedly asks for microphone permission.** Because the stream resets periodically, the browser will prompt for microphone access again. Just click **Allow** each time it appears.

## Browser Compatibility

| Browser | Speech Recognition | Manual Mode |
|---------|-------------------|-------------|
| Chrome / Edge | ✓ Full support | ✓ |
| Firefox | ✗ Limited | ✓ |
| Safari | ✗ Limited | ✓ |

Speech recognition requires Chrome or Edge. All other browsers can still be used in manual grading mode (✓/✗ buttons).
